GESTION ECRAN - LOGICIEL WINDOWS

Description :
Ce logiciel permet d'afficher automatiquement un message plein écran
à une heure définie (ex : 00:00) et de le fermer à une autre heure (ex : 01:00).

Il est conçu pour fonctionner :
- sans droits administrateur
- en arrière-plan
- avec démarrage automatique au lancement de Windows

Fonctionnalités :
- Message personnalisable
- Heures de début et de fin modifiables
- Affichage plein écran
- Fonctionne 24h/24
- Version exécutable (.exe)
- Icône personnalisée

Utilisation :
1. Lancer le fichier gestion_ecran.exe
2. Le programme fonctionne en arrière-plan
3. À l'heure définie, le message s'affiche automatiquement
4. Il se ferme à l'heure de fin définie

Démarrage automatique :
Pour lancer le programme automatiquement avec Windows :
- Appuyer sur Windows + R
- Taper : shell:startup
- Copier le fichier gestion_ecran.exe dans ce dossier

Sécurité :
Ce logiciel ne modifie pas le système,
ne supprime aucun fichier
et ne nécessite aucun droit administrateur.


Langage :
Python (Tkinter), compilé en EXE avec PyInstaller
